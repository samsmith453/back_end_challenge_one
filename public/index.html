How does this thing work?

Using the url in the address bar, or the form below, you can:
  1. Pass a date in string format 'Day Month Year' e.g. 1 January 1970 and we'll show you the unix time stamp for that date.
  2. Pass a unix time stamp and we'll show you the stringy date
  3. Enter some other gobbdley-goop and we'll show the current date and time stamp


Example usage:
  • https://my-unix-timestamp-api.herokuapp.com/5%20Dec%201920
  • https://my-unix-timestamp-api.herokuapp.com/344822400
Example output: { "unix": 1450137600, "natural": "15 December 2015" }
